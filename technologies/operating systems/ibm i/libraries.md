# Libraries
<br />

Create library
* template
    * crtlib lib({{library}})
* example
    * crtlib lib(DBLIB)
<br />
<br />



Delete library
* template
    * dltlib lib({{library}})
* example
    * dltlib lib(DBLIB)
<br />
<br />



Add library
* template
    * `addlible lib({{library}}) position(*FIRST)`
* example
    * `addlible lib(DBLIB) position(*FIRST)`
