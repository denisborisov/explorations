## MySQL

* **Windows** Command prompt / PowerShell console
    * template
        * `select * from {{schema}}.{{table}} \G`
    * example
        * `select * from mysql.user \G`
<br />
<br />

## PostgreSQL

* **Windows** Command prompt / PowerShell console
    * template
        * `select * from {{schema}}.{{table}} \x\g\x`
    * example
        * `select * from pg_catalog.pg_user \x\g\x`
