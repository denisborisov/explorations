# DBMS connection
<br />

## DB2 for i

* **Windows** 5250 Emulator provided by IBM i Access Client Solutions
    * connect
        * `strsql`
    * disconnect
        * <F3> / <F12>
<br />
<br />

## MySQL

* **Windows** Command prompt / PowerShell console
    * connect
        * template
            * `mysql -u {{user name}} -p`
            * `exit`
        * example
            * `mysql -u root -p`
    * disconnect
        * `exit`
<br />
<br />

## PostgreSQL

* **Windows** Command prompt / PowerShell console
    * connect
        * template
            * `psql -U {{user name}}`
        * example
            * `psql -U postgres`
    * disconnect
        * `exit`
<br />
<br />

## SQL Server

* **Windows** Command prompt / PowerShell console
    * connect
        * template
            * `sqlcmd -S {{server}}\{{instance}}`
        * example
            * `sqlcmd -S (localdb)\mssqllocaldb`
    * disconnect
        * `exit`
