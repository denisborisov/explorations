# How to execute OS command.

## DB2 for i

\[Windows\] 5250 Emulator provided by IBM i Access Client Solutions.

* There is no way to do it.



## MySQL

\[Windows\] Command prompt or PowerShell console

There are two possible options:
* `\! {{OS command}}`
    * `\! dir`
* `system {{OS command}}`
    * `system dir`



## PostgreSQL

\[Windows\] Command prompt or PowerShell console

* `\! {{OS command}}`
    * `\! dir`
