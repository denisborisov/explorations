# Execute OS command from SQL CLI
<br />

## DB2 for i

* No way to do it.
<br />
<br />

## MySQL

* **Windows** Command prompt / PowerShell console
    1. connect to DBMS
    2. execute command
        * 1st option
            * template
                * `\! {{OS command}}`
            * example
                * `\! dir`
        * 2nd option
            * template
                * `system {{OS command}}`
            * example
                * `system dir`
<br />
<br />

## PostgreSQL

* **Windows** Command prompt / PowerShell console
    1. connect to DBMS
    2. execute command
        * template
            * `\! {{OS command}}`
        * example
            * `\! dir`
<br />
<br />

## SQL Server

* **Windows** Command prompt / PowerShell console
    1. connect to DBMS
    2. execute commands
        * template
            1. `exec xp_cmdshell '{{command}}'`
            2. `go`
        * example
            1. `exec xp_cmdshell 'dir'`
            2. `go`

P.S. There is an [instruction](https://docs.microsoft.com/en-us/sql/database-engine/configure-windows/xp-cmdshell-server-configuration-option?view=sql-server-ver15) about how to use `xp_cmdshell` if it is turned off.
