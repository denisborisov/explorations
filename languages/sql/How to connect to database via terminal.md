# How to connect to a database via terminal

## DB2 for i

\[Windows\] 5250 Emulator provided by IBM i Access Client Solutions.

`strsql`



## MySQL

\[Windows\] Command prompt or PowerShell console

`mysql -u {{user_name}} -p`

e.g. `mysql -u root -p`



## PostgreSQL

\[Windows\] Command prompt or PowerShell console

`psql -U {{user_name}}`

e.g. `psql -U postgres`
