# How to connect to a database via terminal

## DB2 for i

\[Windows\]
5250 Emulator provided by IBM i Access Client Solutions.
* `strsql`



## MySQL

\[Windows\]
Command prompt or in PowerShell console
* `mysql -u root -p`



## PostgreSQL

\[Windows\]
Command prompt or in PowerShell console
* `psql -U postgres`
