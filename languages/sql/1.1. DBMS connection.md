# DBMS connection
<br />

## DB2 for i

* **Windows** 5250 Emulator provided by IBM i Access Client Solutions
    * `strsql`
<br />
<br />

## MySQL

* **Windows** Command prompt / PowerShell console
    * template
        * `mysql -u {{user name}} -p`
    * example
        * `mysql -u root -p`
<br />
<br />

## PostgreSQL

* **Windows** Command prompt / PowerShell console
    * template
        * `psql -U {{user name}}`
    * example
        * `psql -U postgres`
<br />
<br />

## SQL Server

* **Windows** Command prompt / PowerShell console
    * template
        * `sqlcmd -S {{server}}\{{instance}}`
    * example
        * `sqlcmd -S (localdb)\mssqllocaldb`
